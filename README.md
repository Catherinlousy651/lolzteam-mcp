## 🛠️ Initial setup

Once you install the software, you must link it to your account. Open the application from your desktop shortcut. The program window appears on your screen. You will see a prompt requesting your account credentials.

Input your Lolzteam username and your private API token into the text fields. You can find your API token in your account settings on the official Lolzteam website. Keep this token private and do not share it with others. The software stores this information securely on your device to maintain your session. Click the Save button to apply your settings.

## 🚀 How to use in Claude

The tool works as a bridge. Once you configure the settings, you can open Claude. When you type a query related to the market, the software identifies the intent and fetches the relevant answer.

For example, you can ask, "Show me the current price floor for this item type on Lolzteam." The tool retrieves the data and sends it to the AI. Claude then presents the answer to you in a readable format. You no longer need to verify the info manually on the website.

## 🛡️ Privacy and security

The creators designed this tool with privacy in mind. It acts only as a relay to pass information between your clipboard and the AI interface. It does not store your passwords on a remote server. All data stays local on your Windows machine. If you ever want to reset the tool, you can remove the folder from your Application Data path.

## 🔍 Troubleshooting common issues

If you encounter a problem, check these steps first:

* Application does not open: Ensure you have the latest Windows updates. Restart your computer and try again.
* Data not showing in Claude: Verify that your API token is still valid. You can check this by logging into your account on the main website. If your token expired, generate a new one in your profile settings and update the tool.
* Connection errors: Check your firewall settings. Sometimes, security software blocks new applications from accessing the internet. Add an exception for the lolzteam-mcp executable file in your firewall control panel.
* Updates: The software checks for updates automatically when you launch it. If a new version exists, a window will appear. Follow the prompts to update to the latest release. This ensures you keep access to all market features as the website updates its own interface.
